Abstract

Recent feed-forward 3D reconstruction models, such as VGGT, have proven competitive with traditional optimization-based reconstructors while also providing geometry-aware features useful for other tasks. Here, we show that the quality of these models scales predictably with model and data size. We introduce VGGT-Ω, which substantially improves reconstruction accuracy, efficiency, and capabilities for both static and dynamic scenes. To enable training at an unprecedented scale, we introduce architectural changes that improve training efficiency, a high-quality data annotation pipeline that supports dynamic scenes, and a self-supervised learning protocol.

We simplify VGGT's architecture by using a single dense prediction head with multi-task supervision and removing expensive high-resolution convolutional layers. We also use registers to aggregate scene information into a compact representation, and introduce register attention, which restricts cross-frame information exchange to these registers, in part replacing global attention. VGGT-Ω uses only ∼30% of the GPU memory of its predecessor during training, which lets us train with 15× more supervised data and leverage vast amounts of unlabeled video. VGGT-Ω achieves strong results across multiple static and dynamic benchmarks — e.g., improving the previous best Sintel camera AUC@3° by 77%. We also show that the learned registers can improve vision-language-action models and support alignment with language, suggesting that reconstruction can be a powerful and scalable proxy task for spatial understanding.

Overview

VGGT-Ω is a single feed-forward Transformer that ingests one or more images and jointly predicts camera parameters and per-pixel depth for every frame. Compared to VGGT, the model is leaner, scales further, and handles dynamic content out of the box.

Register attention

A subset of global-attention layers exchanges information only through a small set of scene tokens. The registers carry global scene information and reduce global-attention cost by ~25% FLOPs with no measurable accuracy drop.

One dense head

The expensive DPT blocks running above 1/4 resolution are replaced by an MLP + pixel-shuffle. Tracks and point maps are kept only as training-time losses, freeing memory without losing supervision.

Dynamic-aware data pipeline

A 5-stage pipeline (VLM filter → dynamic mask → multi-method matching → COLMAP + filtering → multi-view consistency) turns 40M internet videos into ~0.8M high-quality annotated sequences, ~1/3 dynamic.

Teacher–student self-supervision

An EMA teacher supervises a student through different augmentations and frame orderings, learning from 18M unlabeled videos to improve out-of-distribution generalization.
